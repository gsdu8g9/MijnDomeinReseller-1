Command: transfer_list

Omschrijving

Toon een overzicht van de domeinverhuizingen uit het account

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
status Toon enkel verhuizingen met de status (tot maximaal 30 dagen terug), waarden: cancelled, pending, transferred Nee
domein Toon enkel verhuizingen van de opgegeven domeinnaam (zonder TLD) (tot maximaal 30 dagen terug) Nee
tld Toon enkel verhuizingen van domeinnamen met de opgegeven TLD (tot maximaal 30 dagen terug) Nee


Uitvoer parameters

Parameter Waarde
transfercount Numerieke waarden van het aantal verhuizingen
 
transfer_id[n] TransferID
domein[n] Domeinnaam
status[n] Status van de verhuizing
status_melding[n] Statusmelding van de verhuizing
datum_invoer[n] Datum waarop de verhuizing werd aangevraagd
datum_update[n] Datum waarop de verhuizing voor het laatst werd bijgewerkt
n wordt vervangen door een opeenlopend cijfer, beginnend bij 0, eindigt bij transfercount
 
errcount 0
