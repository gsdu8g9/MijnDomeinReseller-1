Command: domain_renew

Omschrijving

Verleng een domeinnaam handmatig

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
domein Domeinnaam Ja
tld TLD (extensie) van domeinnaam Ja
duur Numerieke waarde voor aantal te verlengen jaren Ja


Uitvoer parameters

Parameter Waarde
errcount 0
done true


Opmerkingen

1. Aan het verlengen van een domeinnaam zijn kosten verbonden, bekijk de kosten in uw account.
