Command: tld_list

Omschrijving

Toon een overzicht van beschikbare TLD's (extensies)

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja


Uitvoer parameters

Parameter Waarde
tldcount Numerieke waarden van het aantal TLD's
 
tld[n] TLD
n wordt vervangen door een opeenlopend cijfer, beginnend bij 0, eindigt bij tldcount
 
errcount 0
done true
