Command: dns_get_details

Omschrijving

Toon de dnsrecords van een domeinnaam welke de nameservers van MijnDomeinReseller gebruikt

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
domein Domeinnaam Ja
tld TLD (extensie) van domeinnaam Ja


Uitvoer parameters

Parameter Waarde
ttl Time To Live (TTL) van de domeinnaam
recordcount Nummerieke waarde van het aantal dnsrecords
 
record_id[n] recordID van dnsrecord
type[n] Type van het record
host[n] Hostnaam van het record
address[n] Adres van het record
priority[n] Port van het record (enkel voor SRV/MX)
weight[n] Weight van het record (enkel voor SRV)
port[n] Port van het record (enkel voor SRV)
n wordt vervangen door een opeenlopend cijfer, beginnend bij 0, eindigt bij recordcount
 
errcount 0
done true
