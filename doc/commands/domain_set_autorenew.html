Command: domain_set_autorenew

Omschrijving

Schakel de autorenew in of uit

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
domein Domeinnaam Ja
tld TLD (extensie) van domeinnaam Ja
 
autorenew Autorenew instellen, waarden: true (inschakelen) / false(uitschakelen) Ja
 
registrant_approve Toestemming van registrant voor opheffen domeinnaam (enkel bij .DE extensie), waarden: true (toestemming) / false (geen toestemming) Ja, als extensie is .DE en autorenew=false
 


Uitvoer parameters

Parameter Waarde
errcount 0
done true


Opmerkingen

1. Wanneer een .DE domeinnaam zonder toestemming van de houder wordt opgeheven dan wordt de domeinnaam naar de Denic transit gepusht.
Zie voor meer informatie: https://transit.secure.denic.de/en/
