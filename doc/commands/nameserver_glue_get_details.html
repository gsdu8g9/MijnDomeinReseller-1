Command: nameserver_glue_get_details

Omschrijving

Toon een overzicht van de glue records uit het account

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
domein Domeinnaam Ja
host Hostnaam Ja
 


Uitvoer parameters

Parameter Waarde
active 1 voor actief, 0 voor niet actief
 
ipcount Numerieke waarde van het aantal ipadressen
 
ip[n] ipadres
n wordt vervangen door een opeenlopend cijfer, beginnend bij 1, eindigt bij ipcount
 
errcount 0
done true
