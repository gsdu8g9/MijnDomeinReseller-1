Command: domain_auth_info

Omschrijving

Verzend de authorizatiekey voor de verhuizing of trade van een .BE domeinnaam naar de houder of genereer een .EU authorizatiekey

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
domein Domeinnaam Ja
tld TLD (extensie) van domeinnaam Ja
 


Uitvoer parameters

Parameter Waarde
errcount 0
done true


Opmerkingen

1. Een authorizatiekey voor .BE domeinnaam blijft na verzending zeven dagen geldig.
2. Een authorizatiekey voor .EU domeinnaam blijft 40 dagen geldig.
