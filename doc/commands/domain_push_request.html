Command: domain_push_request

Omschrijving

Verhuizen van een bestaande domeinnaam welke in een ander account bij MijnDomeinReseller is ondergebracht (interne verhuizing)

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
domein Domeinnaam voor verhuizing Ja
tld TLD (extensie) van domeinnaam Ja
 
authkey Authorizatiekey voor verhuizing domeinnaam Ja
 


Uitvoer parameters

Parameter Waarde
errcount 0
done true


Opmerkingen

1. Bij een push blijven de instellingen van de domeinnaam (nameservers, contacten, verloopdatum) behouden.
